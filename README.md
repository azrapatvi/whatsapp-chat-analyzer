## How It Works

1. Upload your WhatsApp chat exported as a `.txt` file.
2. Select the user to analyze (or "overall" for group analysis).
3. Click **Show Analysis** to explore detailed statistics and visualizations.

The app automatically generates:
- Graphs for messages over time
- Bar charts for top users and common words
- Pie charts for emoji distribution
- Heatmaps for weekly activity
